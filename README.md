# Projeto Petshop 🦴🐾
O presente projeto trata-se de uma landing page feita para um petshop fictício focado em conversão. 

## Características do desenvolvimento </>
* Desenvolvido em Nextjs
* Utiliza de componentização
* Pastas organizadas e organização de código
* Informações de SEO para melhor rankeamento do site

## Frontend do Projeto 📱 💻
* Desenvolvido com a abordagem Mobile-first
* Possui chamadas para ação com links que levam até o whatsapp
* Possui uma seção dedicada à relados dos clientes (Passando maior credibilidade)
* Interface simples, intuitiva de 100% responsiva
